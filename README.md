# 🥘 Refrigerator Chef

### *Smart recipes from what you already have*

---

## 🧠 About

**Refrigerator Chef** is a smart web application that helps you cook meals using only the ingredients you already have at home.

Just select what's in your fridge — and the app instantly shows you recipes that require no more than 2 missing ingredients.

**No food waste. No last-minute store runs. Just practical, real cooking.**

---

## ✨ Features

- ✅ Ingredient‑based search — no typing, just click
- 🧠 Smart matching — shows recipes missing ≤2 ingredients
- 🥗 Automatic ingredient categorisation (dairy, vegetables, bakery, etc.)
- 🍳 Step‑by‑step instructions with video links (RUTUBE)
- 📱 Works on desktop and mobile
- 🐍 Built with Python (Flask)

---
